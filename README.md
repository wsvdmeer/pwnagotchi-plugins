# Pwnagotchi Plugins

Collection of custom Pwnagotchi plugins for enhanced functionality.

## 🆕 New: BLE IP Broadcasting & Web Bluetooth Finder

Find your Pwnagotchi's IP address in seconds using Web Bluetooth!

**Quick Start:**

1. Update the `bt-tether-helper` plugin (includes BLE broadcasting)
2. Host the [Web Bluetooth IP Finder](web-bluetooth-ip-finder/) on GitHub Pages
3. Scan for your Pwnagotchi from any Chrome/Edge/Opera browser

**📚 Documentation:**

- [⚡ Quick Start Guide](QUICKSTART.md) - Get up and running in 3 minutes
- [📖 Complete Setup Guide](SETUP_GUIDE.md) - Detailed instructions for everything
- [🔧 Git Setup Help](GIT_SETUP.md) - Git commands and GitHub Pages setup

## Plugins

### bt-tether-helper

Comprehensive Bluetooth tethering plugin with web interface, automatic reconnection, and **BLE IP broadcasting**.

**[📖 Read full documentation →](bt-tether-helper/README.md)**

**Quick Features:**

- 🌐 Web UI for easy connection management
- 🤝 Automatic pairing with guided setup
- 📊 On-screen status indicators
- 🔍 Device scanning and discovery
- 🆕 **BLE IP broadcasting for easy discovery**
- 🔄 Auto-reconnect on connection drops

### 🆕 web-bluetooth-ip-finder

Web application that uses Web Bluetooth API to find and display your Pwnagotchi's IP address.

**[📖 Read full documentation →](web-bluetooth-ip-finder/README.md)**

**Quick Features:**

- 🔍 One-click scanning for Pwnagotchi devices
- 📋 Copy IP address to clipboard
- 📱 Works on desktop and Android
- 🎨 Modern, responsive interface
- 🔒 Runs entirely in your browser (no backend)

**Host it yourself:** Follow the setup steps above

### rtc-datetime

Simple plugin to display current time and date on the Pwnagotchi screen.

**[📖 Read full documentation →](rtc-datetime/README.md)**

**Quick Features:**

- ⏰ Customizable time/date format
- 🕐 Uses system RTC time
- 📍 Configurable screen position
- 🌍 Includes timezone setup guide
